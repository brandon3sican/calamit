PROJECT 2

CALaMI-T Cordillera Administrative Landslide Monitoring Integrated Toolbox

Project Overview

Project CALaMI-T is a landslide mitigation researc and development for BLISTT. It aims to develop a landslide movement model toolbox that can predict landslides and provide information on the impact of rain-induced landslides on communities, infrastructure, and the environment.

It is the Project 2 under the Program MULAT BLISTT: Modelling and Understanding LAndslide evenTs in Baguio-La Trinidad-Itogon-Sablan-Tuba-Tublay.

Project Goals

Soil Moisture Threshold

Determine soil moisture threshold values and other parameters for landslide.

Landslide Movement Model Toolbox Software

Develop or adopt a suitable landslide movement model toolbox software.

Landsilde Management Information System

Develop and testing of a landslde management information system.

Mobile Application for Landslide

Develop a mobile application for landslide warning and awareness, reporting and routing to evacuation areas or safe spots.

Early Warning System

Develop a Web and SMS based early warning system with DRRMO Operation Center Dashboard.
